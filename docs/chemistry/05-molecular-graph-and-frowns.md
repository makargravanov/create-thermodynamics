# Граф молекулы и FROWNS

## Внутренний граф

Код: `molecule/graph.rs`

`MolecularStructure` хранит:

- атомы;
- связи;
- исходный код структуры, если он есть;
- топологию, если она нужна для старых структур.

Внутри модели водороды явные. Это центральное правило.

## Почему явные водороды важны

Если водород нужен реакции, он должен быть атомом в графе. Генератор не имеет права “предположить”, что водород где-то есть.

Это делает состояние тяжелее, но честнее:

- масса считается из настоящих атомов;
- валентность проверяется прямо;
- удаление водорода меняет граф и массу;
- генератор не может незаметно создать невозможный продукт.

## FROWNS

Код: `molecule/frowns.rs`

Главные функции:

- `parse_frowns`
- `write_frowns`
- `canonical_structure_code`

Внешняя запись может не содержать нейтральные водороды, но после разбора они становятся явными атомами.

## Редактор графа

Код: `MolecularEditor`

Редактор нужен органическим генераторам. Он умеет:

- добавлять атом;
- удалять атом;
- удалять группу атомов;
- заменять атом;
- менять порядок связи;
- добавлять группу;
- соединять структуры;
- вставлять мостиковый атом;
- разделять структуру по связи.

После изменения граф должен пройти проверку связности, валентности, массы и заряда.

## Функциональные группы

Код: `molecule/functional_group.rs`

Поиск функциональных групп работает по фактическому графу. Он не использует скрытые водороды.
