# Документация проекта

Это папка документации. Ее можно открыть в Obsidian как хранилище.

Главная карта химической модели: [[chemistry/00-index|Химическая модель]].

## Как открыть

1. Открыть Obsidian.
2. Выбрать `Open folder as vault`.
3. Указать папку `docs` в корне репозитория.

Документация хранится рядом с кодом, поэтому ссылки ведут прямо на файлы реализации.
