# Kotlin Migration Notes

The MDK Java template has been replaced with a multi-module Kotlin layout.

Current defaults intentionally keep the original template metadata until the project has final mod identity values:

- `mod_id=create_thermodynamics`
- `mod_group_id=dev.makargravanov.create_thermodynamics`
- `mod_name=create-thermodynamics`

Update `config/mod.properties` when those values are decided.
